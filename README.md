# Employee Attrition Data Analysis

This project analyzes an HR employee dataset to identify key factors influencing employee attrition.

## Dataset
HR Employee Attrition dataset containing 1,470 employee records.

## Objectives
- Identify factors influencing employee turnover
- Analyze relationships between employee attributes and attrition
- Generate insights to support HR decision making

## Analysis Performed
- Descriptive statistics
- Data visualization (histograms, box plots, scatter plots)
- Correlation analysis

## Key Insights
- Higher attrition among younger employees (20–30 years)
- Low income strongly linked to higher turnover
- Long commuting distance increases attrition
- Poor work-life balance and overtime contribute to employee exits

## Tools Used
- Excel
- Statistical Analysis
- Data Visualization
